# Build Photo Call Sheet

Create the call sheet only from confirmed or clearly labeled production facts.

## Required sections

- Project, shoot date, timezone, production address and map coordinates
- General call, crew/talent-specific calls, estimated wrap
- Producer/on-set lead, emergency contact and nearest appropriate emergency facility
- Parking, load-in, access, Wi-Fi/cell, restrooms, holding/changing and accessibility
- Weather summary and decision trigger
- Schedule rows with start/end, duration, activity/setup, location, people, dependencies and notes
- Meal/rest breaks, travel/company moves, golden-hour or hard-light windows, client review and backup plan

## Scheduling rules

1. Start from hard constraints: location hours, talent windows, natural light, deliveries, meals and travel.
2. Add setup/strike time separately from capture time.
3. Group similar setups and schedule must-have shots first.
4. Include 10–20% buffer; never hide over-capacity.
5. Validate that no person, space or critical asset is double-booked.
6. Label personal calls separately when a full contact sheet is not appropriate.

Use `build_call_sheet` for deterministic schedule math, then review local labor and meal rules. Distribute revisions with a version number and timestamp.
