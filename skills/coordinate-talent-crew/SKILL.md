# Coordinate Talent and Crew

Build a roster from the work, not from a generic crew list.

## Workflow

1. Derive roles from shot complexity, lighting, styling, set, data capture, client review, safety and location needs.
2. For each role define scope, decision rights, call/wrap, prep deliverables, rate basis, overtime, expenses, kit fee, cancellation terms and booking status.
3. For talent record casting criteria, fitting, usage, exclusivity, agency, guardian/chaperone needs, accessibility and release status.
4. Map dependencies and handoffs with a RACI table.
5. Create a privacy-aware contact sheet and an escalation tree.
6. Confirm travel, parking, accommodation, meals, dietary restrictions, changing space and holding area.

## Status vocabulary

Use `proposed`, `hold`, `pending terms`, `confirmed`, `released`, or `declined`. Never describe an unconfirmed person as booked.

Do not make hiring decisions using protected characteristics except legitimate, lawful creative requirements. Flag labor, minor, union and overtime rules for qualified local review.
