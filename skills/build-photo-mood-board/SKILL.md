# Build Photo Mood Board

Create a decision tool, not a collage of attractive images.

## Board structure

Organize references into: hero direction; palette/materials; lighting; composition/lens perspective; casting/pose/expression; wardrobe/HMU; location/set; props; post-production; anti-references.

For every image, annotate the transferable attribute and what must not be copied. Avoid implying that a reference can be reproduced exactly.

## Process

1. Read the creative brief and extract 3–5 visual principles.
2. Define a search vocabulary for each board section.
3. Curate a small, coherent set; remove duplicates and contradictory directions.
4. Record source/creator links and usage status. References guide production; they do not grant reuse rights.
5. Summarize the board as executable rules for light, lens, palette, framing, performance, styling, set and grade.
6. Ask the approver to accept one direction or resolve named conflicts.

## Output

Return a board outline with image slots, search terms, annotations, palette values, visual rules, conflicts, exclusions, and approval question. If actual image sourcing is requested, use approved search tools and preserve attribution links.
