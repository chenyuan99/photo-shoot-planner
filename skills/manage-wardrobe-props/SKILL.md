# Manage Wardrobe and Props

Connect every physical item to a look, shot and responsible owner.

## Workflow

1. Create IDs for looks, garments, accessories, hero products and props.
2. Map each item to talent, setup and shot IDs; record size/color/variant, source, cost, status, owner and return deadline.
3. Plan fittings and approvals; note alterations, backup sizes and duplicates.
4. Pack by setup in labeled garment bags/bins; keep a photo continuity record.
5. Create check-in/check-out and damage/condition logs.
6. Plan racks, steam/iron, mirrors, changing/privacy, cleaning, product security and transport.
7. Flag hazardous, fragile, perishable, trademarked, culturally sensitive or animal-derived items for review.

Never assume visible logos, artwork or distinctive property are cleared. Route permission questions to `$check-shoot-permits-releases`.
